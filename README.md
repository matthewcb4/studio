# fRepo - v2.0.0 (Official MVP)

This is the official MVP release of fRepo (Tonal Tracker).

This version includes a stable, feature-complete application with:
- User Authentication (Email/Password & Google)
- A dynamic Dashboard with stats, a muscle heatmap, and volume charts.
- Full workout creation, editing, and logging functionality.
- AI-powered workout generation.
- Progress tracking for body weight and individual lifts.
- Full PWA (Progressive Web App) support for offline use and installability.

To get started, take a look at src/app/page.tsx.
